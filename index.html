Interview prep
Three questions asked at every anesthesia interview. Practice out loud until natural.
Q1 — "Why didn't you match anesthesia the first time?"
"My application had real strengths — I interviewed at UCSD, Cedars-Sinai, OHSU, and UC Davis. But I had not yet demonstrated sustained clinical performance in a rigorous environment over time. I chose IM specifically to provide exactly that evidence. My performance in IM is my answer."
Q2 — "Why anesthesia — and why now?"
"IM gave me a longitudinal view of how chronic disease evolves. That made me increasingly drawn to the perioperative window — where chronic trajectories become acutely consequential. Anesthesia requires real-time simultaneous management of interacting organ systems under time pressure. That's the cognitive environment where I feel most engaged."
Q3 — "What do you bring that others don't?"
"Three things: perioperative AKI research at a research level, IM-trained longitudinal physiologic reasoning most applicants lack, and operational leadership at scale that maps to perioperative systems thinking."

github personal access token

This token lets the Save buttons push your data directly to GitHub. It's stored only in your browser — never sent anywhere except GitHub's API.

To get a token: github.com → Profile → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → check repo → no expiration → Generate → copy it.

Once set, every Save button pushes to GitHub automatically. No Claude Code needed for logging.

Powers the AI buttons — daily plans, UWorld recommendations, reading lists, interview evaluation. Optional — dashboard works without it.

To get a key: console.anthropic.com → API Keys → Create Key → copy it. Cost: ~$0.003 per button press.
